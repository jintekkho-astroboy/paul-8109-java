At a library, user can borrow books, CD-ROMS and magazines.
Each of those has a different loan duration
- books can be borrowed for 14 days (author, publisher)
- video CD-ROMS can be borrowed for 7 days (rating, length of video)
- magazines can be borrowed for 3 days (publisher, series)
A user can borrow a number of lending materials, there is a due date and a loan number issued.
